## 📌 Overview

**DevMind** is a full-stack AI chat application powered by **Google Gemini API**, designed to deliver a smooth, responsive, and modern chat experience.

Built with scalability and clean UI in mind, this project demonstrates how to integrate LLMs into real-world applications using a **MERN-style architecture** (without React if applicable, or adjust if used).

---

## ✨ Features

* 🤖 AI Chat powered by Gemini API
* ⚡ Real-time conversational responses
* 🎨 Modern UI (Dark theme + gradient design)
* 🔐 Authentication system (Login / Signup)
* 📡 Separate frontend & backend deployment
* 🚀 Optimized performance with clean architecture
* 📱 Responsive design (mobile-friendly)
* 🔔 Toast notifications & UX feedback

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/devmind.git
cd devmind
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
PORT=5000
```

Run server:

```bash
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 API Key Notice

⚠️ This project does **NOT include any API keys** for security reasons.

To run locally, you must:

* Get your Gemini API key from Google AI Studio
* Add it in your `.env` file
