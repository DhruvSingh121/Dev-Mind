🚀 Dev-Mind

Dev-Mind — Where Code Meets Intelligence.

A full-stack AI chat application built using the MERN stack and powered by the Gemini API. Dev-Mind replicates a ChatGPT-like experience with secure authentication, persistent chat history, and developer-friendly features like code highlighting.

✨ Features
💬 Real-time AI chat (Gemini API)
🔐 JWT-based authentication (Login / Signup)
📂 Chat history stored in MongoDB
🎨 Code highlighting for better readability
⚡ Responsive UI with clean CSS
🔄 Dynamic state management with React
🌐 RESTful API architecture (Node.js + Express)

📦 Installation
# Clone the repository
git clone https://github.com/your-username/dev-mind.git

# Navigate to project
cd dev-mind

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install
⚙️ Environment Variables

Create a .env file inside the server folder:

GEMINI_API_KEY=your_api_key

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret

▶️ Run Locally

# Start backend (from server folder)
npm run dev

# Start frontend (from client folder)
npm start
